=head1 NAME

F<epgsearchdirs.conf> – Aufnahmeverzeichnisse

=head1 BESCHREIBUNG

In EPGSearch muss man oft ganze Verzeichnispfade eingeben, etwa
beim Editieren von Suchtimern. Da dies recht mühselig ist, können
in dieser Datei häufig genutzte Pfade vorgegeben werden, die dann
im Menü einfach ausgewählt werden können.

Diese Datei ist eine der Quellen für das Menü C<Verzeichnis wählen>.
Die dort angezeigten Einträge können aus folgenden Quellen stammen:

=over 2

=item * aktuelle Aufnahmeverzeichnisse

=item * aktuelle Timer-Verzeichnisse

=item * in Suchtimern verwendete Verzeichnisse

=item * in dieser Datei angegebene Verzeichnisse

=item * Einträge in der Datei F<folders.conf> des VDR

=back

Das Menü führt vorgefundenen Verzeichnisse zusammen, zeigt also jedes
Verzeichnis nur einmal an. Mit der Taste C<Gelb> lässt sich die
angezeigte Verzeichnisebene ändern. Verzeichniseinträge, die
Variablen für EPG-Kategorien enthalten, wie etwa C<%Genre%>,
erscheinen immer vor anderen Verzeichnissen. Sie sind zudem
unabhängig von der Ebene und werden mit ihrem vollständigen
Verzeichnispfad angezeigt.

Wenn dieses Menü aus dem Timer-Editieren-Menü aufgerufen wird und das
ausgewählte Element bereits die Variablen C<%Title%> oder C<%Subtitle>
enthält, wird der Timer-Eintrag C<Datei> gelöscht, da Titel oder
Untertitel bereits im Eintrag C<Verzeichnis> enthalten sind.

=head1 FORMAT

Jeder Zeile enthält einen Verzeichnispfad, der gegebenenfalls Variablen
enthalten kann.
Verwendet werden können interne Variablen, die Variablen von EPG-Kategorien
aus F<epgsearchcats.conf> sowie die in der Datei F<epgsearchuservars.conf>
konfigurierten Variablen.

Folgende interne Variablen stehen zur Verfügung:

=over 4

=item B<%Time%>

Der Beginn einer Sendung im Format 'HH:MM'.

=item B<%Time_D%>

Das Datum des Beginns einer Sendung im Format 'TT'.

=item B<%Time_W%>

Der Name des Wochentages, an dem eine Sendung beginnt.

=item B<%Time_Lng%>

Datum und Startzeit einer Sendung in I<Epoch>-Notation
(Sekunden seit dem 01.01.1970, 00:00 UTC).

=item B<%Date%>

Das Datum des Beginns einer Sendung im Format 'TT.MM.JJ'.

=item B<%DateSh%>

Das Datum des Beginns einer Sendung im Format 'TT.MM'.

=item B<%Title%>

Der Titel einer Sendung.

=item B<%Subtitle%>

Der Untertitel einer Sendung.

=back

Für weitere Variablen siehe auch B<epgsearchcats.conf>(5) und
B<epgsearchuservars.conf>(5).
