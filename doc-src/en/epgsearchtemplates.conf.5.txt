=head1 NAME

F<epgsearchtemplates.conf> – templates for searches and search timers

=head1 DESCRIPTION

Templates simplify the creation of new searches and search timers.
The templates are saved in this file.

=head1 SYNTAX

This file has the same format as the the file F<epgsearch.conf>.
See B<epgsearch.conf>(5) for details about its syntax.
