=head1 NAME

F<noannounce.conf> – events not to be announced by OSD

=head1 DESCRIPTION

This file contains a events not to be announced by OSD when found
by the search-timer background thread.

If you press one of the keys C<OK> or C<0> to C<9> during an event announcement,
you will be asked whether you want to disable future announcements permanently
(keys C<OK> or C<0>) or just for the next 1 to 9 days (corresponding to the keys
C<1> to C<9>). If you confirm by pressing the key C<OK> again, the setting will
be applied.

=head1 FORMAT

Every line represents an event not to be announced,
comprising the following fields:

=over 4

=item B<1 – Title>

String with the event's title.

=item B<2 – Subtitle ("episode")>

String with the event's subtitle, which often refers to the
episode of a series.

=item B<3 – Channel identifier>

String with the event's channel identifier in VDR notation
(e.g., C<S19.2E-1-1019-10301>).

=item B<4 – Start time>

The event's start date and time in I<Epoch> encoding
(i.e., seconds since 1970-01-01 00:00 UTC).

=item B<5 – Time of next announcement>

The date and time of the event's next announcement in I<Epoch> encoding
(i.e., seconds since 1970-01-01 00:00 UTC).

A value of 0 permanently disables future announcements.

=back

Fields are separated by C<:> characters. Thus, a C<:> character
in title or subtitle will be translated to a C<|> character.
